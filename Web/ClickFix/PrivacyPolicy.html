1. Data controller

ClickFix Mitigator is a browser extension designed to detect ClickFix attack patterns and alert users about risks related to clipboard usage.

2. Data collected

The extension may process and/or send the following data solely for security purposes:

  • Detection events: URL, hostname, date and time (timestamp), detected signals, alert message, and relevant snippets of analyzed content.
  • Usage statistics: number of alerts generated, number of blocks, and extension activation status.
  • Local lists: allowlist and blocklist stored locally in the user's browser.
  • Referrer URL: reference page (referrer) from which a possible ClickFix flow was accessed, if available.

No personal data such as name, email address, or information that directly identifies the user is collected.

3. Purpose of processing

Data is used only to:

  • Detect and block ClickFix attack patterns.
  • Generate security alerts in real time.
  • Improve effectiveness through aggregated statistics.
  • Maintain and optimize allowlist and blocklist lists.
  • Analyze the origin of suspicious redirects to improve incident response.

4. Legal basis

Processing is based on legitimate interest in user security and protection against threats.

5. Data sharing

  • Data may be sent to the ClickFix Mitigator server for processing alerts and statistics.
  • Data is not sold, rented, or shared with third parties, unless strictly necessary to operate the service.

6. Retention

Data is retained only for as long as necessary to fulfill the purposes described and to ensure integrity of security records.

7. Security

Reasonable technical and organizational measures are applied to protect transmitted and stored data against unauthorized access, loss, or alteration.

8. User rights

Users can request information or exercise their rights of access, rectification, or deletion through the contact channels listed on the official site.

9. Policy changes

This policy may be updated periodically. Updates will be published in this document.
